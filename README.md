# AI Portfolio Hub

这是我的 AI 作品集主仓库，用来整理和展示 AI 产品原型、数据分析项目、AI 视频制作流程，以及 Agent / Skill 工具链实验。

## 作品方向

- AI 产品原型：把真实业务问题做成可交互 demo，展示产品判断、页面设计和工程落地能力。
- 数据分析与 Power BI：用模拟业务数据完成指标体系、可视化看板和业务洞察。
- AI 视频工作流：整理从分镜、提示词、关键帧、图生视频到剪辑交付的制作流程。
- Agent / Skill 工具链：把重复流程沉淀成可复用的 Codex skill、自动化脚本或插件。

## 重点项目

| 项目 | 类型 | 状态 | 说明 |
| --- | --- | --- | --- |
| Tenclass AI Demo | AI 产品原型 | 整理中 | 面向教育/企微风控/经营洞察的交互式 demo 与汇报材料 |
| Power BI Interview Dashboard | 数据分析 | 整理中 | 面试用业务分析看板，包含模拟数据、PBIX 和分析说明 |
| AI Video Production Workflow | AI 视频 | 整理中 | AI 广告/故事板制作流程、提示词和关键帧方案 |
| Codex Skill / Pet Lab | Agent 工具链 | 实验中 | 自定义 skill、宠物 spritesheet、自动化流程实验 |
